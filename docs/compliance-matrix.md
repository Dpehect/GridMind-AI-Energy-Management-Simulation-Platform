# GridMind Compliance Matrix

## Identity and access

- Password hashing
- Session expiry and revocation
- Role-based authorization
- Tenant isolation
- Building-level access
- Security event logging

## Data integrity

- Transactions
- Optimistic concurrency
- Soft delete and restore
- Backup checksum validation
- Import deduplication
- Entity versioning

## Reliability

- Health endpoints
- Background-job retries
- Structured logs
- Error grouping
- Diagnostic runs
- Log retention

## Accessibility

Target: WCAG 2.2 AA.

## Security

Controls are aligned with common OWASP application-security practices. Formal certification requires an external audit.
