# GridMind Design System

Phase 02 introduces semantic tokens, reusable primitives, domain components, interaction states, responsive behavior, and accessibility defaults.

## Principles
- Operational clarity before decoration.
- Semantic color communicates system state.
- Motion explains hierarchy and change.
- Dense information remains scannable.
- Light and dark themes are designed independently.

## Component layers
1. Primitive UI: input, label, separator, progress, skeleton.
2. Composite UI: metric card, page header, section card, alert banner.
3. Data UI: table, status badge, filter chip.
4. Interaction UI: segmented control and keyboard affordances.
