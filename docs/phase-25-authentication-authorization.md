# Phase 25 — Authentication & Authorization

Delivered:
- Scrypt password hashing
- Cryptographically random session tokens
- SHA-256 token storage
- HttpOnly, SameSite and Secure cookie configuration
- Database-backed sessions
- Session expiry and revocation
- Login and logout server actions
- Role and permission matrix
- Protected dashboard layout
- Route-level permission layouts
- Permission-enforced mutation actions
- Audit logging for authentication events
- User profile and logout controls
- Seeded local accounts
- Authentication and authorization tests

The implementation remains local-first and requires no external identity provider.
