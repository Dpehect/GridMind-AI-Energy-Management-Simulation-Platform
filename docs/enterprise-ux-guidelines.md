# GridMind Enterprise UX Guidelines

## Navigation

- Group modules by user intent.
- Show only authorized routes.
- Preserve active-route visibility.
- Use breadcrumbs on nested pages.

## Data-heavy screens

Prefer:

- master/detail
- table-first layouts
- filters
- sticky actions
- pagination
- virtualization

Avoid excessive metric-card repetition.

## Mobile

Technician workflows must prioritize:

- current task
- location
- checklist
- parts used
- completion evidence

## Empty states

Every empty state should explain:

1. What is missing.
2. Why it matters.
3. What action to take next.
