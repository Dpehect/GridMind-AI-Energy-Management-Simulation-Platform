# Phase 12 — 3D Energy Digital Twin

The digital twin is rendered locally with React Three Fiber and deterministic building data.

## Capabilities

- Exploded multi-floor building view
- Floor isolation
- Zone-level heat coloring
- Alert markers
- Interactive labels
- Camera presets
- Quality, balanced and economy rendering profiles
- Lazy-loaded route bundle

No hosted 3D service, map provider or API key is required.
