# GridMind Administrator Handbook

## First startup

1. Configure the environment.
2. Create the bootstrap administrator.
3. Start GridMind once.
4. Confirm administrator access.
5. Remove the bootstrap password from the environment.

## User administration

- Create users.
- Assign roles.
- Grant workspace membership.
- Grant building access.
- Disable unused accounts.
- Review active sessions and security events.

## Operations

- Monitor background jobs.
- Review observability incidents.
- Run diagnostics.
- Verify backups.
- Review compliance and release status.

## Production rules

- Demo users must remain disabled.
- HTTPS is mandatory.
- Backups must be copied off-host.
- Restore tests must be performed regularly.
