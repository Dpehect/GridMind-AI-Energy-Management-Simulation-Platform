# GridMind Architecture Overview

## Presentation

Next.js App Router with server components, server actions and responsive client interfaces.

## Domain services

- Energy and analytics
- Buildings, meters and devices
- Maintenance and operations
- Goals and recommendations
- Reports and approvals
- Jobs and automation

## Persistence

Prisma with SQLite for local and single-host deployments.

## Identity

Local users, hashed passwords, database-backed sessions, role permissions, organization memberships and workspace memberships.

## Reliability

Health endpoints, structured logs, error incidents, diagnostics, backup verification and release validation.

## Deployment

Standalone Next.js output, Docker Compose, reverse proxy and systemd examples.
