# GridMind

GridMind is a local-first energy management and simulation platform. Phase 01 establishes the production foundation, application shell, design tokens, dashboard preview, motion infrastructure, validation utilities, and project governance documentation.

## Requirements

- Node.js 20.9 or newer
- npm 10 or newer

## Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality checks

```bash
npm run typecheck
npm run lint
npm run format:check
npm run build
```

## Phase 01 scope

- Premium public landing page
- Responsive application shell
- Dashboard foundation with realistic energy data
- Theme, notification and smooth-scroll providers
- Shared UI primitives and motion tokens
- Strict TypeScript, linting, formatting and security headers
- Architecture, security, contribution and setup documentation

No API key, hosted database, or third-party account is required.

## Phase 02

Open `/design-system` inside the application shell to review GridMind's component language, responsive states, operational metrics and accessibility patterns.
